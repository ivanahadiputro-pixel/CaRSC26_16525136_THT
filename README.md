# Repositori ini disusun sesuai template yang telah diberikan, berikut merupakan penjelasan singkat mengenai setiap directory yang terdapat di repositori ini
1. docs/

    Folder docs berisi dokumentasi utama dalam format Markdown, yang mencakup:
    - Jawaban untuk Soal A dan Soal B
    - Catatan dan log progres pengerjaan

    Folder ini berfungsi sebagai dokumen tertulis utama yang menjelaskan teori, analisis, dan hasil pembelajaran.

2. lampiran/

    Folder lampiran berisi berkas-berkas pendukung, antara lain:
    - File gambar (screenshots, ilustrasi) yang digunakan dan dirujuk dalam file Markdown pada folder docs
    - File tambahan yang diperlukan oleh program Python, seperti file input atau data pendukung

    Folder ini bertujuan untuk memisahkan aset non-kode agar dokumentasi dan program tetap terstruktur.    

3. src/

    Folder src berisi seluruh source code yang telah diimplementasikan, khususnya:
    - Program-program yang berkaitan dengan Soal B
    Implementasi konsep pemrograman 
    File Python atau C++ yang dapat dijalankan secara terpisah

    Folder ini merupakan inti dari implementasi teknis dan pemrograman dalam repository.
